# John Lou L. Cambial — Career Portfolio

A clean, multi-page portfolio website built with HTML, CSS, and vanilla JS.
Cream / gold / charcoal palette. No frameworks, no build step — just open the files.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Home / cover page |
| `about.html` | Who I Am + Goals |
| `resume.html` | Skills Snapshot + Experience |
| `work.html` | Process + Projects |
| `contact.html` | Contact info |
| `style.css` | Shared styles for all pages |

## Add Your Photo

On `index.html`, find the placeholder `<div>` inside `.hero-right` and replace it with:

```html
<img src="photo.jpg" alt="John Lou L. Cambial" class="hero-photo" />
```

Upload your photo as `photo.jpg` to the same folder.

## Publishing to GitHub Pages (Free Hosting)

### Step 1 — Create a GitHub account
Go to https://github.com and sign up if you haven't.

### Step 2 — Create a new repository
1. Click the **+** icon → **New repository**
2. Name it: `portfolio` (or anything you like)
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload your files
1. Click **Add file → Upload files**
2. Drag all 6 files (`index.html`, `about.html`, `resume.html`, `work.html`, `contact.html`, `style.css`) into the upload area
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to your repository → **Settings** tab
2. Scroll down to **Pages** (left sidebar)
3. Under **Source**, select: `Deploy from a branch`
4. Branch: `main` · Folder: `/ (root)`
5. Click **Save**

### Step 5 — Get your live URL
After ~1 minute, your site will be live at:

```
https://YOUR-USERNAME.github.io/portfolio/
```

You can share this link on your LinkedIn, resume, and job applications.

## Optional: Custom Domain
If you want `johncambial.com` instead of `github.io`:
1. Buy a domain (Namecheap, GoDaddy, etc.)
2. In GitHub Pages settings, add your custom domain
3. Point your domain's DNS to GitHub — they provide instructions.
